# Analysis Results

This folder contains screenshots and visual outputs generated from SQL analysis of the Stroke Prediction Dataset.

## Key Findings

### Age Group and Stroke Risk
Patients aged 70+ exhibited the highest stroke rate (17.75%).

### Smoking Status Analysis
Former smokers showed the highest observed stroke rate (7.91%), potentially influenced by higher average age.

### Hypertension Analysis
Patients with hypertension demonstrated a substantially higher stroke prevalence compared to non-hypertensive individuals.

### Gender Analysis
Male patients showed a slightly higher stroke rate than female patients.

### Overall Dataset Distribution
The dataset is highly imbalanced, with stroke cases representing approximately 4.9% of total patients.This folder contains query outputs, charts, and screenshots.
